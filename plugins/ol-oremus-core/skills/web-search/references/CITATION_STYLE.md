# Citation Style

## Required format
- Provide sources at the end of the answer.
- Group sources by the claim they support when there are multiple claims.
- Include publication dates when visible on the page.

## Example
Answer:
- The policy was updated in June 2025.

Sources:
- Policy update (June 2025): https://example.com/policy

## Notes
- Use primary sources whenever possible.
- If a source is paywalled, note it and use an accessible summary.
