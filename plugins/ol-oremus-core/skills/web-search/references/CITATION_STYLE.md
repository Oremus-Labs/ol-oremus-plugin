# Citation Style

## Required format
- Provide sources at the end as a simple list.
- Each entry must be an exact URL that was read via `web_url_read`.

## Example
Answer:
- The policy was updated in June 2025.

Sources:
- https://example.com/policy

## Notes
- Use primary sources whenever possible.
- If a source is paywalled, note it and use an accessible summary.
